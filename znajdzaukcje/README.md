Znajdz aukcje
=============
*versja 1.0 15 Apr 2013

Wtyczka pozwalająca wyszukiwać zaznaczony tekst na polskim serwisie aukcyjnym Allegro.

This plugin allow for searching selected text in popular polish auction portal Allegro.
